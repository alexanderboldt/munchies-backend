# Munchies

This is a backend application made with Kotlin and Spring Boot for managing recipes.

## Tech-Stack

### Development
- Kotlin
- Spring Boot
- Hibernate
- MySQL
- Keycloak
- Flyway
- OpenFeign
- OpenApi

### Test
- RestAssured
- Mockito
- Kotest
- Docker-Testcontainer

### Buildsystem
- Gradle
